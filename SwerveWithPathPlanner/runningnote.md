# TO RUN CODE: 
- Turn on access point
- Turn on robot

## While Running:
- New button: Hold B to align to hub, and it will turn for shuttling as well.
- Make sure to run auto at beginning to reset POSE2D for hub alignment to work.
    - Example: turn on teleop, align robot to hub. Click auto, make sure New Auto is selected, and after it resets position, teleop is good to go.
